Приложение использует синхронизацию с LocalStorage для хранения контактов в памяти браузера. Все действия сохраняются и при перезагрузке страницы всё останется, как прежде.

Кнопка "Очистить всё", очищает Ваш LocalStorage на данном ресурсе.
